## Point YAML

Required fields: canonical_topic, project_role, objective, audience, mode, question, title, point, teachback_expected, domain_core, project_lens, model, essentials, proves, does_not_prove, implications, caveats, exclusions, glossary, source_map, and visual_priority.

## Synthesis review

Synthesis JSON records domain_query_sha256, prior_sha256, baseline_sha256, source_model_sha256, artifact_sha256, and model_sha256. The model prior also records the exact domain_query_sha256, model_identity, status=coverage_hypothesis, project_context_seen=false, and is_evidence=false. Its scores are domain_baseline_coverage, project_relevance, and synthesis_gain.

Hard fields:

- prior_project_context_seen=false
- baseline_project_context_seen=false
- source_baseline_context_seen=false
- prior_used_as_evidence=false
- authoritative_baseline=true
- domain_first=true
- project_value_statement_present=true
- blind_comparison_completed=true
- final_domain_coverage_not_below_baseline=true
- final_project_fidelity_not_below_source=true
- baseline_strengths_adopted contains at least one item
- source_strengths_adopted contains at least two items
- baseline critical essentials all selected as must and mapped to final_ids
- at least two source-model project insights selected
- unresolved_critical_domain_gaps=0
- domain_core_share >= 0.50
- assessment_share <= 0.20

Every review JSON contains round, artifact_sha256, model_sha256, critical_issues, and issues. The gate rejects stale reviews. Fact review also records complete claim, model, and freshness coverage with zero critical unsupported claims, conflicts, or authoritative-source gaps.

For a structural or security topic, the Domain review also records the required flags and their result: context_dependencies_closed, cross_boundary_results_authenticated, threat_assumptions_explicit, trust_semantics_not_collapsed, and decision_result_identifies_evidence_and_policy. Required checks must be true when the topic contains freshness, identity, binding, a security boundary, or a decision-bearing result.

Gate JSON is generated by pipeline.py and must not be edited.
